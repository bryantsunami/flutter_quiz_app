# quiz_app

A new Flutter project.

## Background

This app was created by following along with the Flutter & Dart Udemy course by "Maximilian".

It covers a lot of basic flutter principles such as:

- Basic syntax
- Class constructors
- How the Widget Tree works
- Layout
- Functions and event handling
- Stateful vs Stateless Widgets
- Private properties
- Initial Styling concepts
- Maps and Mapping Lists
- Splitting widgets
